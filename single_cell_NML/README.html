NeuroML2 description of a reduced-morphology layer V pyramidal cell.
Christoph Metzner & Tuomo Mäki-Marttunen, 2015-2016

Files included in the NeuroML2 implementation:
 CaDynamics_E2_NML2.nml                       #Ca2+ dynamics nml file
 CaDynamics_E2_NML2_apicTrunk.nml             #Ca2+ dynamics (using parameters for
                                              #  the apical trunk) nml file
 CaDynamics_E2_NML2_apicTuft.nml              #Ca2+ dynamics (using parameters for
                                              #  the apical tuft) nml file
 CaDynamics_E2_NML2_soma.nml                  #Ca2+ dynamics (using parameters for
                                              #  the soma) nml file
 Ca_HVA.channel.nml                           #High-voltage activated Ca2+ current nml file
 Ca_LVAst.channel.nml                         #Low-voltage activated Ca2+ current nml file
 Ih.channel.nml                               #HCN channel current nml file
 Im.channel.nml                               #Muscarinic K+ current nml file
 K_Pst.channel.nml                            #Persistent K+ current nml file
 K_Tst.channel.nml                            #Transient K+ current nml file
 NaTa_t.channel.nml                           #Transient Na+ current nml file
 Nap_Et2.channel.nml                          #Persistent Na+ current nml file
 SK_E2.channel.nml                            #SK current nml file
 SKv3_1.channel.nml                           #Kv3.1 K+ current nml file
 pas.channel.nml                              #Passive current nml file
 Hay_Cell_Condition1.net.nml                  #Nml file defining the network (of one neuron)
 LEMS_RedL5bPyrCellHayEtAl2011-Condition1.xml #LEMS file defining the
                                              #  stimulus for condition 1
 Soma_AllCML.cell.nml                         #Nml file defining the neuron morphology
                                              #  and parameters for ion channel
                                              #  conductances and Ca2+ dynamics

The nml files are based on the implementation given in
https://github.com/OpenSourceBrain/L5bPyrCellHayEtAl2011
